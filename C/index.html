C pre začiatočníkov

Stránka na učenie jazyka C s inštaláciou, základmi, knižnicami a 30 otázkovým testom.

Inštalácia Základy Knižnice PONG Test

1. Inštalácia nástrojov

Windows

  1. Stiahni si MinGW alebo MSYS2 pre kompilátor GCC.
  2. Pridaj binárku do PATH a over gcc --version.
  3. Použi VS Code s rozšírením C/C++ od Microsoft.

Linux

  1. Inštaluj GCC: sudo apt install build-essential alebo ekvivalent.
  2. Otvor VS Code a nainštaluj rozšírenie C/C++.
  3. Skontroluj kompilátor: gcc --version.

iPad

  1. Použi webové prostredie s podporou C, napr. repl.it alebo GitHub Codespaces.
  2. Pracuj s VS Code Web alebo alternatívou na písanie a kompiláciu v cloude.
  3. Upravuj súbory a spúšťaj kompiláciu cez vzdialený terminál.

2. Základy C

Prvý program

Základný program v C vypíše text pomocou printf.

#include 

int main(void) {
    printf("Ahoj, C!\n");
    return 0;
}

Premenné a typy

C vyžaduje deklaráciu typu pre každú premennú.

int cislo = 10;
float desatinne = 3.14f;
char znak = 'A';
char text[] = "Ahoj";

printf("%d %f %c %s\n", cislo, desatinne, znak, text);

Podmienky a cykly

Odsadenie je pre čitateľnosť, bloky delí stredník a zátvorky.

if (cislo > 5) {
    printf("Veľké číslo\n");
} else {
    printf("Malé číslo\n");
}

for (int i = 0; i < 3; i++) {
    printf("%d\n", i);
}

while (cislo > 0) {
    printf("%d\n", cislo);
    cislo--;
}

Funkcie

Funkcie v C definujeme s návratovým typom.

int sucet(int a, int b) {
    return a + b;
}

int main(void) {
    printf("%d\n", sucet(3, 4));
    return 0;
}

stdio.h

    printf("Ahoj\n");

}

stdlib.h

Funkcie pre pamäť, konverziu a ukončenie programu.

int main(void) {
    int *p = malloc(sizeof(int) * 3);
    if (!p) return 1;
    free(p);
    return 0;
}
