Pygame a hra PONG

Učiť sa Pygame s jednoduchou hrou PONG: inštalácia, základné moduly, herný cyklus a logika.

Späť na Python

1. Čo je Pygame?

Pygame je knižnica pre Python, ktorá sprístupňuje grafiku, zvuk a vstup pre tvorbu 2D hier. Umožňuje pracovať s oknom, udalosťami a kreslením v reálnom čase.

2. Inštalácia a prvé okno

Inštalácia Pygame:

pip install pygame

Základný kód pre otvorenie okna:

import pygame

pygame.init()  # Inicializuje všetky Pygame moduly
screen = pygame.display.set_mode((800, 600))  # Nastaví okno na šírku 800 a výšku 600
pygame.display.set_caption("PONG")  # Názov okna
running = True
while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
    screen.fill((0, 0, 0))  # Vymaže obrazovku čiernou farbou
    pygame.display.flip()  # Aktualizuje obsah okna
pygame.quit()  # Ukončí Pygame

Vysvetlenie:

  • pygame.init() spúšťa interné moduly Pygame, ako zvuk a vstup.
  • pygame.display.set_mode((800, 600)) vytvorí zobrazovaciu plochu s rozmermi v pixeloch.
  • pygame.display.set_caption() nastaví názov okna.
  • pygame.event.get() vráti zoznam udalostí, napr. stlačenie tlačidla alebo zatvorenie okna.
  • screen.fill((0, 0, 0)) vyplní obrazovku RGB farbou: čierna = (0, 0, 0).
  • pygame.display.flip() vykreslí zmeny do okna.

3. Základné objekty v Pygame

Najčastejšie použité prvky v hre:

  • Surface – obrázok alebo plocha, na ktorú kreslíme.
  • Rect – obdĺžnik s polohou a veľkosťou (x, y, width, height).
  • Clock – objekt pre nastavenie snímkovej frekvencie.
  • Event – údaje o vstupoch, ukončení alebo stlačení kláves.

Príklad vytvorenia tlačidla a renderovania textu:

player = pygame.Rect(30, 250, 10, 100)
clock = pygame.time.Clock()
font = pygame.font.SysFont("Arial", 24)
text_surface = font.render("Skóre: 0", True, (255, 255, 255))

Vysvetlenie:

  • pygame.Rect(30, 250, 10, 100) vytvorí obdĺžnik so súradnicami x=30, y=250, šírka=10, výška=100.
  • pygame.time.Clock() vytvorí meradlo času pre ovládanie FPS.
  • pygame.font.SysFont("Arial", 24) vyberie font Arial vo veľkosti 24 pixelov.
  • font.render(text, True, color) vytvorí obrázok textu, kde True znamená hladký anti-aliasing.

4. Algoritmus hry PONG

Hra PONG má tieto hlavné kroky:

  1. Inicializovať Pygame a vytvoriť okno.
  2. Vytvoriť objekty hráča, protivníka a lopty.
  3. Spracovať vstupy od hráča (pohyb rakety).
  4. Aktualizovať pozíciu lopty a skontrolovať kolíziu so stenami a raketami.
  5. Pridať body, keď lopta prejde za raketu.
  6. Vykresliť všetky objekty na obrazovku.
  7. Zopakovať cyklus, kým hráč neukončí hru.

Pseudokód herného cyklu:

while hra_bezi:
    spracuj_udalosti()
    pohyb_hraca()
    pohyb_oponenta()
    pohyb_lopty()
    kontrola_kolizii()
    vykresli_obrazovku()
    cakanie_na_dalsiu_framu()

5. Krok za krokom: implementácia PONG

Tu je kompletný príklad s vysvetlením každej časti:

import pygame
import sys

pygame.init()  # Inicializuje Pygame moduly

WIDTH, HEIGHT = 800, 600
screen = pygame.display.set_mode((WIDTH, HEIGHT))
pygame.display.set_caption("PONG")

WHITE = (255, 255, 255)
BLACK = (0, 0, 0)

player = pygame.Rect(30, 250, 10, 100)  # ľavá raketa
opponent = pygame.Rect(760, 250, 10, 100)  # pravá raketa
ball = pygame.Rect(WIDTH // 2 - 15, HEIGHT // 2 - 15, 30, 30)  # lopta

player_speed = 0
opponent_speed = 5
ball_speed = [4, 4]  # rýchlosť lopty v osiach x a y

clock = pygame.time.Clock()
font = pygame.font.SysFont("Arial", 30)

player_score = 0
opponent_score = 0

running = True
while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
        if event.type == pygame.KEYDOWN:
            if event.key == pygame.K_w:
                player_speed = -6
            if event.key == pygame.K_s:
                player_speed = 6
        if event.type == pygame.KEYUP:
            if event.key == pygame.K_w or event.key == pygame.K_s:
                player_speed = 0

    player.y += player_speed  # pohyb hráčovej rakety
    if player.top <= 0:
        player.top = 0
    if player.bottom >= HEIGHT:
        player.bottom = HEIGHT

    opponent.y += opponent_speed  # jednoduchá AI logika pre protivníka
    if opponent.top <= 0 or opponent.bottom >= HEIGHT:
        opponent_speed *= -1

    ball.x += ball_speed[0]
    ball.y += ball_speed[1]

    if ball.top <= 0 or ball.bottom >= HEIGHT:
        ball_speed[1] *= -1  # odraz od hornej alebo dolnej steny

    if ball.colliderect(player) or ball.colliderect(opponent):
        ball_speed[0] *= -1  # odraz od rakety

    if ball.left <= 0:
        opponent_score += 1
        ball.center = (WIDTH // 2, HEIGHT // 2)
        ball_speed[0] *= -1

    if ball.right >= WIDTH:
        player_score += 1
        ball.center = (WIDTH // 2, HEIGHT // 2)
        ball_speed[0] *= -1

    screen.fill(BLACK)
    pygame.draw.rect(screen, WHITE, player)
    pygame.draw.rect(screen, WHITE, opponent)
    pygame.draw.ellipse(screen, WHITE, ball)
    pygame.draw.aaline(screen, WHITE, (WIDTH // 2, 0), (WIDTH // 2, HEIGHT))

    player_text = font.render(f"Hráč: {player_score}", True, WHITE)
    opponent_text = font.render(f"Rival: {opponent_score}", True, WHITE)
    screen.blit(player_text, (20, 20))
    screen.blit(opponent_text, (WIDTH - opponent_text.get_width() - 20, 20))

    pygame.display.flip()
    clock.tick(60)

pygame.quit()
sys.exit()

Vysvetlenie:

  • pygame.Rect(x, y, width, height) definuje obdĺžnikový objekt v priestore.
  • player.y += player_speed posúva raketu podľa rýchlosti.
  • ball.colliderect(player) kontroluje, či sa lopta dotýka rakety.
  • ball_speed[0] *= -1 zmení smer pohybu v osi x, čím lopta odrazí.
  • pygame.draw.rect(screen, WHITE, player) nakreslí biely obdĺžnik na obrazovku.
  • pygame.draw.ellipse(screen, WHITE, ball) nakreslí bielu loptičku.
  • pygame.display.flip() zobrazí výsledok kreslenia na obrazovke.
  • clock.tick(60) udržiava hru na približne 60 snímkach za sekundu.

6. Parametre a význam

Najdôležitejšie parametre v tomto kóde:

  • (800, 600) – rozmery okna v pixeloch.
  • (255, 255, 255) – biela farba v RGB.
  • player_speed = -6 – záporná hodnota posúva nahor, kladná nadol.
  • ball_speed = [4, 4] – rýchlosť lopty v x a y smere; prvý prvok je horizontálna rýchlosť, druhý je vertikálna.
  • WIDTH // 2 – celočíselné delenie pre stred obrazovky.
