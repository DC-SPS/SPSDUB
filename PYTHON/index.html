Čo je Python?

Python je vysokoúrovňový, interpretovaný programovací jazyk vytvorený Guido van Rossumom v roku 1989. Je známy svojou jednoduchou a čitateľnou syntaxou, ktorá mu umožňuje byť ideálnym jazykom pre začiatočníkov aj skúsených vývojárov.

Kde sa Python používa?

  • Webový vývoj - Django, Flask
  • Analýza dát - Pandas, NumPy, Matplotlib
  • Umelá inteligencia a strojové učenie - TensorFlow, PyTorch
  • Automatizácia - skriptovanie a DevOps
  • Vedecké výpočty - SciPy
  • Vývoj hier - Pygame

2️⃣ Základy Pythonu s detailnými vysvetleniami

Prvý program: print()

Funkcia print() vypíše text alebo iné hodnoty na obrazovku (do konzoly).

print("Ahoj, Python!")
print("2 + 3 =", 2 + 3)
print("Python je skvelý!")

Výstup:

Ahoj, Python!
2 + 3 = 5
Python je skvelý!

Vysvetlenie: Reťazec v úvodzovkách je text, ktorý sa vypíše. Čiarka oddeluje jednotlivé položky, ktoré sa vypíšu s medzerou medzi nimi.

Premenné a dátové typy

Premenná je kontajner na uloženie dát. Na rozdiel od iných jazykov sa v Pythone typ automaticky určuje.

meno = "Anna"           # str (reťazec)
vek = 25               # int (celé číslo)
vyska = 1.72           # float (desatinné číslo)
je_programator = True  # bool (logická hodnota: True/False)

print(meno, vek, vyska, je_programator)
print(type(vek))       #

Hlavné dátové typy:

Typ Príklad Popis
int 42, -5, 0 Celé čísla bez desatinnej časti
float 3.14, -2.5, 0.0 Desatinné čísla
str "Ahoj", 'text' Textové reťazce v úvodzovkách
bool True, False Logické hodnoty (pravda/nepravda)
list [1, 2, 3] Zoznam hodnôt v hranatých zátvorkách
dict {"meno": "Anna"} Slovník s kľúčmi a hodnotami

Podmienky (if, elif, else)

Podmienky umožňujú spustiť rôzny kód na základe splnenia podmienky.

vek = 18

if vek < 13:
    print("Si dieťa")
elif vek < 18:
    print("Si tínedžer")
else:
    print("Si dospelý")

Operátory porovnávania:

Operátor Popis Príklad
== Rovná sa x == 5
!= Nerovná sa x != 5
< Menšie ako x < 10
> Väčšie ako x > 3
<= Menšie alebo rovné x <= 10
>= Väčšie alebo rovné x >= 0

Cykly (for a while)

Cykly opakujú blok kódu viacerí krát, kým je splnená podmienka alebo pokým sa neprejde cez všetky prvky.

For cyklus - iterácia cez zoznam:

# Cyklus cez čísla 0 až 4
for i in range(5):
    print(f"Číslo: {i}")

# Cyklus cez zoznam
ovocie = ["jablko", "banán", "hruška"]
for ovocko in ovocie:
    print(f"Milujem {ovocko}")

While cyklus - opakujúci sa kód:

pocet = 0
while pocet < 5:
    print(f"Počet: {pocet}")
    pocet += 1  # Zvýš pocet o 1

Funkcie

Funkcia je opakujúci sa blok kódu, ktorý môžeš použiť viackrát bez potreby opakovania kódu.

def pozdrav(meno):
    """Funkcia, ktorá pozdraví používateľa."""
    return f"Ahoj, {meno}!"

# Volanie funkcie
vysledok = pozdrav("Mária")
print(vysledok)  # Ahoj, Mária!

# Funkcia s viacerými parametrami
def zrataj(a, b):
    return a + b

print(zrataj(3, 5))  # 8

Dôležité koncepty funkcií:

  • def - slovo na definovanie funkcie
  • meno - názov funkcie (bez medzier, malé písmená)
  • (parametre) - vstupné hodnoty
  • return - návratová hodnota (čo funkcia vracia)
  • """docstring""" - dokumentácia funkcie

Zoznamy (Lists)

Zoznam je zbierka prvkov v poradí, ktoré sa dá meniť.

ovocie = ["jablko", "banán", "hruška", "pomaranč"]

# Prístup k prvkom
print(ovocie[0])      # jablko (prvý prvok)
print(ovocie[-1])     # pomaranč (posledný prvok)
print(len(ovocie))    # 4 (počet prvkov)

# Pridanie prvku
ovocie.append("kivi")

# Odstránenie prvku
ovocie.remove("banán")

# Iterácia cez zoznam
for ovocko in ovocie:
    print(ovocko)

Slovníky (Dictionaries)

Slovník ukladá dáta ako páry kľúč-hodnota (ako telefonný zoznam).

osoba = {
    "meno": "Ján",
    "vek": 30,
    "mesto": "Bratislava",
    "zamestnanie": "programátor"
}

# Prístup k hodnotám
print(osoba["meno"])      # Ján
print(osoba.get("vek"))   # 30

# Zmena hodnoty
osoba["vek"] = 31

# Pridanie nového kľúča
osoba["telefon"] = "0905123456"

# Iterácia cez slovník
for kluc, hodnota in osoba.items():
    print(f"{kluc}: {hodnota}")

3️⃣ Vstavané moduly Pythonu

Modul je balík funkcií a tried, ktoré sa dajú naimportovať a použiť v tvojom programe.

📐 math - Matematika

Matematické funkcie a konštanty.

import math

# Funkcie
print(math.sqrt(16))    # 4.0
print(math.pow(2, 3))   # 8.0
print(math.ceil(3.2))   # 4 (zaokrúhlenie hore)
print(math.floor(3.9))  # 3 (zaokrúhlenie dole)

# Konštanty
print(math.pi)          # 3.14159...
print(math.e)           # 2.71828...

📁 os - Operačný systém

Práca so súborovým systémom a cestami.

import os

# Aktuálny adresár
print(os.getcwd())

# Zoznam súborov
print(os.listdir('.'))

# Vytvorenie priečinka
os.mkdir('novy_priečinok')

# Overenie či existuje
if os.path.exists('súbor.txt'):

🎲 random - Náhodné hodnoty

import random

# Náhodné číslo od 1 do 10

# Náhodná voľba z zoznamu
farba = random.choice(

# Náhodné desatinné číslo 0-1
hodnota = random.random()

karty = [1, 2, 3, 4, 5]

⏰ datetime - Dátum a čas

Práca s dátumami a časmi.

from datetime import datetime, timedelta

# Aktuálny dátum a čas
teraz = datetime.now()
print(teraz)

# Formátovanie
text = teraz.strftime("%d.%m.%Y %H:%M:%S")
print(text)

# Rozdiel v čase
zitra = teraz + timedelta(days=1)
print(f"Zajtra: {zitra.date()}")

📄 json - JSON formát

import json

# Python slovník
osoba = {"meno": "Ján", "vek": 30}
